# RTTTL Player

An online RTTTL ringtone player. Browse, search, edit, and play over 11,000 classic ringtones directly in your browser — no installation required.

## Features

- Browse and search a collection of 11,000+ RTTTL ringtones
- Play tones using the Web Audio API (no plugins needed)
- Edit RTTTL code in real-time and hear changes instantly
- Create your own custom tones
- Save favorites to local storage
- Dark / light / system theme
- English and Traditional Chinese (zh-TW) interface

## Data Source

All ringtone data is sourced from the [PICAXE RTTTL collection](https://picaxe.com/rtttl-ringtones-for-tune-command/) and is used for non-commercial, educational purposes only. The original data remains the property of its respective authors.
